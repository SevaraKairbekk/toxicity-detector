\# Smart Toxicity Detector 🧠



Анализ текста на токсичность с использованием ИИ. Определяет явные и скрытые формы агрессии, оскорблений, угроз и контента, вредящего репутации.



\## 🚀 Возможности



\- ✅ Определение явных оскорблений

\- 🔍 Распознавание скрытой агрессии и сарказма

\- ⚠️ Обнаружение угроз

\- 🔞 Фильтрация нецензурной лексики

\- 📊 Детальный анализ по 5 категориям токсичности



\## 🛠️ Технологии



\- \*\*Frontend\*\*: React

\- \*\*Backend\*\*: Node.js, Express

\- \*\*AI Model\*\*: rubert-tiny-toxicity (Apache 2.0)
